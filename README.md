# 📊 Social Media Engagement Prediction System

## 📌 Project Overview
This project focuses on predicting the engagement level of social media posts using Machine Learning.  
The system analyzes factors such as caption length, hashtags, posting time, and follower count to classify posts into:
- High Engagement
- Medium Engagement
- Low Engagement

It helps content creators, influencers, and businesses make data-driven decisions before posting content.

---

## 🎯 Objectives
- Analyze factors affecting social media engagement  
- Build a Machine Learning classification model  
- Predict engagement levels (High/Medium/Low)  
- Develop a simple and user-friendly interface  

---

## 🧠 Machine Learning Approach
- **Problem Type:** Classification  
- **Model Used:** Random Forest Classifier  

### Features Used:
- Post Type  
- Caption Length  
- Hashtags Count  
- Post Day  
- Post Hour  
- Follower Count  

### Target Variable:
- Engagement Category (High / Medium / Low)
